# My Website

A personal portfolio website featuring projects, services, consulting, hackathons, and ideas.

## Features

- **Home** - Landing page with introduction
- **Projects** - Showcase of personal projects
- **Services** - Services offered
- **Consulting** - Consulting information
- **Hackathons** - Hackathon participation and achievements
- **Ideas** - Collection of ideas
- **Contact** - Contact information and form

## Files

- `index.html` - Home page
- `projects.html` - Projects page
- `services.html` - Services page
- `consulting.html` - Consulting page
- `hackathons.html` - Hackathons page
- `ideas.html` - Ideas page
- `contact.html` - Contact page
- `script.js` - JavaScript functionality
- `styles.css` - Styling
- `assets/` - Images and other assets

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Navigate through the website using the menu
